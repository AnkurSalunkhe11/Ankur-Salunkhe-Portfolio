## Overview

A premium, typography-first developer portfolio design. It uses a high-contrast dark foundation (deep slate) with a split accent language based on active domain tabs: rich indigo for Computer Science / AI, and minty emerald for Mechanical Engineering / CFD.

## Styling Principles

1. **Understated Borders:** Eliminate heavy background fills or shadows. Use ultra-thin borders `1px solid rgba(255, 255, 255, 0.05)` (dark) and `rgba(0, 0, 0, 0.05)` (light) to container boundaries.
2. **Dynamic Segments:** Sliding Framer Motion segment pill tabs in the header to transition between CS and Mechanical domain contexts.
3. **Card Grids:** Bento-like grids for skills and projects featuring high-contrast text layout elements and subtle transition scales.
